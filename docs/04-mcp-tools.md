# 04. MCP Tools — Контракт

> Статус: ⬜ Не начато

## Цель

Определить набор MCP-инструментов, которые агенты используют для взаимодействия с памятью, поиском и средой. Стандартизация через Model Context Protocol.

## Инструменты

### Память (CRUD)

| Tool            | Описание                        | Параметры                                 |
| :-------------- | :------------------------------ | :---------------------------------------- |
| `memory_read`   | Чтение записи из памяти по ID   | `id`, `layer?`                            |
| `memory_write`  | Создание/обновление записи      | `layer`, `content`, `tags[]`, `agent_id?` |
| `memory_delete` | Удаление записи                 | `id`                                      |
| `memory_search` | Гибридный поиск (FTS5 + vector) | `query`, `layer?`, `limit?`, `agent_id?`  |

### Логирование

| Tool         | Описание                    | Параметры                                               |
| :----------- | :-------------------------- | :------------------------------------------------------ |
| `log_append` | Append в Слой 4 (сырой лог) | `session_id`, `content`, `type` (user/assistant/system) |
| `log_read`   | Чтение логов за сессию      | `session_id`, `limit?`                                  |

### Embeddings

| Tool           | Описание                      | Параметры                       |
| :------------- | :---------------------------- | :------------------------------ |
| `embed_text`   | Получить embedding текста     | `text`, `model?` (text/code)    |
| `embed_search` | Векторный поиск по similarity | `query`, `top_k?`, `threshold?` |

### Rerank

| Tool     | Описание                          | Параметры                       |
| :------- | :-------------------------------- | :------------------------------ |
| `rerank` | Переранжировать результаты поиска | `query`, `passages[]`, `top_n?` |

### Утилиты

| Tool               | Описание                                  | Параметры    |
| :----------------- | :---------------------------------------- | :----------- |
| `context_summary`  | Получить Executive Summary текущей сессии | `session_id` |
| `compress_history` | Сжать историю чата в саммари              | `messages[]` |

## Формат ответов

Все инструменты возвращают JSON:

```json
{
  "success": true,
  "data": { ... },
  "error": null
}
```
