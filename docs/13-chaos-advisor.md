# 13 — Chaos Advisor: Mistral как генератор инициативы

## Проблема

Claude (Opus/Sonnet) по натуре **осторожный и реактивный** — ждёт задачу, не лезет вперёд.
В автономном режиме это приводит к "сделал минимум → done". Даже с промптом
про проактивность — Opus всё равно склонен к conservative behavior.

## Решение

**Mistral (NVIDIA NIM, бесплатный)** как "советник хаоса" — дешёвый, дерзкий,
безбашенный. Перед каждой автономной сессией (или когда задач нет) Teamlead
спрашивает Mistral: "Что бы ты сделал?". Mistral генерирует идеи, а Teamlead
фильтрует и выполняет.

## Архитектура

```
Scheduler (каждые 30 мин)
    │
    ├─ [1] Teamlead (Claude) получает задачу
    │   └─ Если задач нет или "всё сделано":
    │       │
    │       ▼
    │   [2] consult_chaos() → Mistral (NVIDIA, 0 RPM cost)
    │       └─ "Вот контекст о Дмитрии. Предложи 3 конкретных действия
    │           которые можно сделать прямо сейчас. Будь дерзким и креативным."
    │       │
    │       ▼
    │   [3] Teamlead выбирает лучшую идею → выполняет
    │
    └─ Результат → TG + memory
```

## Реализация

### Новый tool: `consult_chaos`

Вызывает Mistral через NVIDIA NIM (бесплатный, 40 RPM) с промптом:
- Контекст: что агент уже делал сегодня (из memory)
- Вопрос: "Что делать дальше? Предложи 3 идеи."
- Модель: `nvidia/mistral-nemo-minitron-8b-8k-instruct` или chaos из MODEL_MAP

### Промпт для Mistral (Chaos Advisor)

```
Ты — генератор идей для AI-агента. Агент работает на фрилансера Дмитрия (22 года, 
Nuxt/TypeScript/PHP стек, ищет доход).

Агент уже сделал сегодня: {recent_actions}
Текущее время: {time}, день: {weekday}

Предложи 3 КОНКРЕТНЫХ действия, которые агент может сделать ПРЯМО СЕЙЧАС.
Требования:
- Каждое действие должно быть выполнимо за 5-10 минут
- Используй интернет (парсинг, поиск, анализ)
- Будь дерзким и нестандартным — агент осторожный, ему нужен пинок
- Не повторяй то, что уже сделано
- Формат: номер + действие + почему это полезно

НЕ предлагай: "проверь почту", "посмотри чаты" — это и так делается.
Предлагай: конкретные сайты, конкретные запросы, конкретные действия.
```

### Интеграция в system-prompt

Добавить в автономный промт:
```
### Chaos Advisor:
Если ты не знаешь что делать, или все основные задачи выполнены — 
вызови `consult_chaos`. Это дешёвый быстрый советник, который предложит 
идеи. Ты НЕ обязан следовать всем — выбери лучшую и действуй.
Стоимость: 0 RPM (NVIDIA NIM, бесплатно). Не стесняйся спрашивать.
```

### Когда вызывать

1. В начале сессии, если нет явной задачи
2. После выполнения всех основных действий (перед `done`)
3. Когда агент "застрял" — не знает что искать в интернете

## Преимущества

- **Бесплатно** — NVIDIA NIM, не тратит GitHub Copilot RPM
- **Быстро** — Mistral отвечает за 1-2 секунды
- **Разнообразие** — каждый раз другие идеи (temperature 0.9)
- **Безопасно** — Teamlead (Claude) фильтрует бред, выполняет только разумное
- **Не блокирует** — advisory only, Claude всегда имеет последнее слово

## Альтернативы (если Mistral не подходит)

- `chaos` из MODEL_MAP (gpt-5.4-mini) — тоже дешёвый
- DeepSeek через OpenRouter — хорош для кода
- Qwen через NVIDIA — хорош для рассуждений
