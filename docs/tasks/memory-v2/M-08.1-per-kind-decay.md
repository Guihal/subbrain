**Tier:** P2 · **Effort:** S · **Deps:** M-08 (forgetting curve) + M-07 (kind enum) — landed · **Status:** DONE

## Цель

M-08 положил MemoryBank-style `R = exp(-Δt / tau)` в RAG ranking, с persona override (skipPersona = R=1.0). Все остальные kinds (semantic, episodic, procedural) затухают одинаково. Per CoALA / cognitive science:
- **persona** — never decay (already done).
- **semantic** (general facts) — default decay rate (baseline).
- **episodic** (events, session-bound) — faster decay (more transient by nature).
- **procedural** (skills, "how-to") — slower decay (skill knowledge persists).

После M-08.1: `computeRecallScore` принимает `kind` parameter; tau умножается на per-kind multiplier:
- persona → R=1.0 (skip, как сейчас).
- semantic → tau × 1.0 (default).
- episodic → tau × 0.5 (faster decay).
- procedural → tau × 2.0 (slower decay).

Env knobs: `RAG_DECAY_MULT_EPISODIC` (default 0.5), `RAG_DECAY_MULT_PROCEDURAL` (default 2.0). Read at call-time.

## Изменение

```ts
// packages/core/src/lib/memory-decay.ts
export type MemoryKindForDecay = "persona" | "semantic" | "episodic" | "procedural" | undefined;

const DECAY_MULT_EPISODIC = () => Number(process.env.RAG_DECAY_MULT_EPISODIC) || 0.5;
const DECAY_MULT_PROCEDURAL = () => Number(process.env.RAG_DECAY_MULT_PROCEDURAL) || 2.0;

export function computeRecallScore(
  nowSeconds: number,
  lastAccessSeconds: number | null,
  accessCount: number,
  salience: number,
  kind?: MemoryKindForDecay,
): number {
  if (lastAccessSeconds === null) return 1.0;
  const dt = Math.max(0, nowSeconds - lastAccessSeconds);
  const baseStrengthDays = 1 + Math.log(1 + accessCount);
  const salienceFactor = 0.5 + Math.max(0, Math.min(1, salience));
  let kindMult = 1.0;
  if (kind === "episodic") kindMult = DECAY_MULT_EPISODIC();
  else if (kind === "procedural") kindMult = DECAY_MULT_PROCEDURAL();
  // persona handled by skipPersona override in applyForgettingCurve (R=1.0).
  // semantic + undefined → kindMult=1.0 (baseline).
  const tau = baseStrengthDays * salienceFactor * kindMult * 86400;
  return Math.exp(-dt / tau);
}
```

`applyForgettingCurve` уже передаёт `r.kind` если бы оно было. Need verify в M-08 implementation — если нет, добавить (`r.kind` уже на RAGResult от M-07).
