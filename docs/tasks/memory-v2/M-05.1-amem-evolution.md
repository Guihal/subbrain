### `evolveNeighbour` shape

```ts
const MAX_TAGS_DEFAULT = 10;

function evolveNeighbour(
  memory: MemoryDB,
  neighbourId: string,
  neighbourLayer: "context" | "shared",
  insertedTags: string[],
  maxTags: number,
): void {
  if (insertedTags.length === 0) return;

  const row =
    neighbourLayer === "context"
      ? memory.getContext(neighbourId)
      : memory.getShared(neighbourId);
  if (!row) return;

  const currentTags = parseTags(row.tags ?? "");
  const merged = mergeUnique(currentTags, insertedTags, maxTags);
  if (sameSet(currentTags, merged)) return; // no-op

  const newTagsCsv = merged.join(",");
  if (neighbourLayer === "context") {
    memory.updateContext(neighbourId, { tags: newTagsCsv });
  } else {
    memory.updateShared(neighbourId, { tags: newTagsCsv });
  }
}
```

Helpers (also module-private, вЙ§30 LOC –≤—Б–µ–≥–Њ):
- `parseTags(csv)` вАФ split `,`, trim, drop empty.
- `mergeUnique(a, b, cap)` вАФ preserve order; drop dups; tail-truncate to cap (keep newest).
- `sameSet(a, b)` вАФ order-insensitive equality.

### Edge cases handled

- **insertedTags empty** вАФ early return, no DB writes.
- **neighbour deleted between linkEdge –Є evolveNeighbour** вАФ getContext/getSharedById returns null вЖТ early return.
- **merged === current** вАФ no-op (sameSet check), avoid spurious updated_at bumps.
- **>MAX_TAGS already on neighbour** вАФ merged includes new tags, then tail-truncated; oldest tags dropped.
- **evolveNeighbour throw** вАФ caught + logged, doesn't abort linkRelated (same pattern as edge insert error).
