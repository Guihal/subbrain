## Порядок исполнения

`04 → 02 → 01 → 03`:
- **04 (UI)** — быстрый, снимает ежедневное раздражение.
- **02 (TG polling)** — дешёвый, сразу даёт бытовую пользу (напоминания).
- **01 (RAG для отчётов)** — когда отчёты уже стабильно выходят, обогащаем их.
- **03 (freelance)** — последним: нужен поднятый web-стэк из 04, максимальный риск банов, большой объём.
